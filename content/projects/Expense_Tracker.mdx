## Overview

Expense Tracker is a modern personal finance management application designed to help users track incomes, expenses, and budgets through intuitive dashboards and analytics.

The platform provides secure authentication, interactive visualizations, and a responsive premium user experience across all devices.

## Features

- Real-time income and expense tracking
- Interactive analytics dashboards using Recharts
- Budget planning and financial insights
- Secure JWT and Google OAuth authentication
- Responsive UI built with Tailwind CSS
- Smooth animations using Framer Motion

## Architecture

The frontend is built with React and Tailwind CSS, while the backend uses Node.js and MongoDB for secure data storage and authentication.

Recharts powers financial visualizations for real-time spending analysis.

## Results

- Improved financial tracking with intuitive dashboards
- Responsive cross-device user experience
- Secure cloud-based authentication and data management
