## Overview

Rocket Rider is an interactive browser game that uses webcam-based hand gesture recognition for real-time gameplay control.

The project combines computer vision, immersive sound design, and responsive UI elements to deliver an engaging gaming experience directly in the browser.

## Features

- Real-time hand gesture recognition using MediaPipe
- Gesture-based gameplay controls
- Responsive game overlays and UI animations
- Immersive sound effects using Tone.js
- Cross-device browser compatibility

## Architecture

MediaPipe Tasks Vision processes live webcam input to identify hand gestures and translate movements into gameplay actions.

The frontend is powered by React, Next.js, and TypeScript for a seamless interactive experience.

## Results

- Real-time low-latency gesture interaction
- Enhanced gameplay immersion through audio feedback
- Smooth browser-based experience across devices
