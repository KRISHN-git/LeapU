## Overview

Vehicle Damage Detector is an end-to-end AI platform that analyzes uploaded vehicle images to detect damages, classify severity, and generate automated repair insights.

The system integrates computer vision and generative AI to streamline vehicle inspection workflows.

## Features

- Vehicle damage detection using YOLO
- Vehicle classification using TensorFlow
- Automated repair summaries with Gemini API
- Safety assessment and repair cost estimation
- Interactive analytics dashboards
- Secure image storage and authentication

## Architecture

The platform combines a React frontend with a Flask backend for image processing and inference workflows.

Cloudinary manages image storage while MongoDB stores user history and analytics.

## Results

- Automated multi-angle vehicle damage analysis
- Reduced manual inspection effort through AI summarization
- Secure scalable cloud-based deployment
